# 💰 Expense Tracker

Expense Tracker is a simple web application that helps users track their income and expenses. It allows users to add transactions and see their financial balance in an easy and organized way.

## 🚀 Features
- Add income and expense transactions
- View transaction history
- Automatic balance calculation
- Simple and user-friendly interface
- Responsive design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- React

## 📌 How to Run the Project

1. Clone the repository
git clone https://github.com/yourusername/expense-tracker.git

2. Navigate to the project folder
cd expense-tracker

3. Install dependencies
npm install

4. Run the project
npm start

## 📊 Project Purpose
This project was built to practice frontend development and learn how to manage application state and user inputs.
